# What I Learned

In this assignment I learned how to use Git and GitHub to manage a project.
I learned how to create branches to work on different features separately,
and then merge them all back into the main branch when done.

I also learned basic networking concepts like IP addresses, subnet masks,
and how to divide a network into smaller subnets using CIDR notation.

Using Git locally and then pushing to GitHub showed me how version control
works in a real development workflow.
